## 1. 小说上传与解析

### 1.1 功能概述

用户上传小说文本文件，系统自动解析并提取角色、场景、分镜等信息。

### 1.2 核心类

| 类名 | 文件 | 职责 |
|------|------|------|
| `NovelService` | src/services/NovelService.cpp | 小说上传、数据库操作 |
| `AnalysisService` | src/services/AnalysisService.cpp | 分析流程调度 |

### 1.3 实现流程

```
用户上传文件
     ↓
NovelService::uploadNovel()
     ├── 读取文件内容
     ├── 创建小说记录
     └── 保存到数据库
     ↓
AnalysisService::startAnalysis()
     ├── 调用 AI 分析角色
     ├── 调用 AI 分析场景
     ├── 生成分镜脚本
     └── 保存分析结果
```

### 1.4 关键代码

```cpp
// NovelService.cpp - 小说上传
bool NovelService::uploadNovel(const QString& title, const QString& content)
{
    Novel novel;
    novel.setTitle(title);
    novel.setContent(content);
    novel.setCreatedAt(QDateTime::currentDateTime());
    
    return DatabaseManager::instance()->insert("novels", novel.toVariantMap());
}

// AnalysisService.cpp - 开始分析
void AnalysisService::startAnalysis(const QString& novelId)
{
    QString content = fetchNovelContent(novelId);
    
    // 1. 提取角色
    QList<Character> characters = CharacterExtractor::instance()->extract(content);
    saveCharacters(novelId, characters);
    
    // 2. 提取场景
    QList<Scene> scenes = SceneExtractor::instance()->extract(content);
    saveScenes(novelId, scenes);
    
    // 3. 生成分镜
    QList<Panel> panels = StoryboardService::instance()->generate(novelId, content);
    savePanels(novelId, panels);
    
    emit analysisCompleted(novelId);
}
```

### 1.5 技术要点

- 文件读取使用 `QFile` 和 `QTextStream`
- 大文件分块读取，避免内存溢出
- 分析过程异步执行，不阻塞 UI

| `BibleImageService` | src/services/BibleImageService.cpp | 圣经图片生成 |

### 2.3 实现流程

```
小说文本
     ↓
CharacterExtractor::extract()
     ├── AI 提取角色列表
     ├── 提取外观描述
     ├── Enrich 补充缺失信息
     └── 保存到数据库
     ↓

     ├── 构建角色描述 Prompt
     ├── 调用图像生成 API
     ├── 下载并保存图片
     └── 更新角色记录
```

### 2.4 关键代码

```cpp
// CharacterExtractor.cpp - 角色提取
QList<Character> CharacterExtractor::extract(const QString& text)
{
    // 1. 调用 AI 提取角色
    QJsonObject result = QwenClient::instance()->chat(buildExtractPrompt(text));
    
    // 2. 解析 AI 返回
    QJsonArray charactersJson = result["characters"].toArray();
    
    QList<Character> characters;
    for (const QJsonValue& val : charactersJson) {
        ExtractedCharacter extracted = parseCharacter(val.toObject());
        
        // 3. Enrich 补充缺失信息（只在字段为空时推断）
        enrichCharacterAppearanceFromText(extracted, text);
        
        characters.append(extracted.toCharacter());
    }
    
    return characters;
}

// Enrich 逻辑 - 只填空不覆盖
void enrichCharacterAppearanceFromText(ExtractedCharacter& extracted, const QString& context)
{
    if (extracted.hairColor.isEmpty()) {
        const QString textHairColor = matchFirstCapture(context, hairColorPatterns);
        if (!textHairColor.isEmpty()) {
            extracted.hairColor = textHairColor;
        }
    }
    // 其他字段同理...
}

// BibleImageService.cpp - 圣经图片生成
void BibleImageService::generateBibleImages(const QString& novelId)
{
    QList<Character> characters = getCharactersByNovel(novelId);
    
    for (const Character& ch : characters) {
        // 构建角色描述 Prompt
        QString prompt = PromptBuilder::buildCharacterPrompt(ch.toJson(), options);
        
        // 调用图像生成 API
        QImage image = QwenImageClient::instance()->generate(prompt);
        
        // 保存图片
        QString path = saveImage(image, ch.name());
        ch.setPortraitPath(path);
        
        updateCharacter(ch);
    }
}
```

### 2.5 技术要点

- **Enrich 只填空不覆盖**：AI 返回的字段优先级最高，文本推断只补充空字段
- **正则表达式优化**：避免误匹配（如"白色短袖"中的"白色"不应匹配为发色）
- **批量生成**：支持批量生成所有角色的圣经图片

```cpp
// ImageService.cpp - 面板图片生成
ImageService::GenerateResult ImageService::generatePanelImageCore(
    const QString& panelId, 
    const ResolutionConfig& resolution,
    bool useReference,
    const EditHint& hint)
{
    // 1. 获取面板数据
    Panel panel = fetchPanel(panelId);
    
    // 2. 构建 Prompt
    GenerationContext ctx;
    ctx.panel = panel;
    buildPromptForPanel(ctx, panel.toJson());
    
    // 3. 加载参考图
    if (useReference) {
        loadReferenceImage(ctx);
    }
    
    // 4. 选择引擎并生成
    GenerateResult result;
    if (shouldUseQwen(ctx)) {
        if (ctx.hasReferenceImage()) {
            result = QwenImageClient::instance()->edit(buildQwenEditOptions(ctx, resolution));
        } else {
            result = QwenImageClient::instance()->generate(buildQwenGenerateOptions(ctx, resolution));
        }
    } else {
        result = VolcEngineImageClient::instance()->generate(buildVolcEngineOptions(ctx, resolution));
    }
    
    // 5. 保存结果
    if (result.success) {
        QString path = saveImage(result.image, panelId);
        updatePanelImage(panelId, path);
    }
    
    return result;
}

// PromptBuilder.cpp - 构建 Prompt
bool ImageService::buildPromptForPanel(GenerationContext& ctx, const QJsonObject& panelJson)
{
    // 1. 获取角色和场景参考
    QString novelId = fetchNovelIdByStoryboardId(ctx.panel.storyboardId());
    QMap<QString, QJsonObject> characterRefs = fetchCharacterRefs(novelId, ctx.referenceImages, ctx.characterRefPaths);
    QMap<QString, QJsonObject> sceneRefs = fetchSceneRefs(novelId, ctx.referenceImages);
    
    // 2. 解析主要角色
    ctx.primaryCharacterName = resolvePrimaryCharacter(panelJson);
    
    // 3. 生成 Seed（同章节使用相同种子）
    ctx.generationSeed = generateSeedFromStoryboardId(ctx.panel.storyboardId());
    
    // 4. 构建 Prompt
    PromptBuilder::PromptResult result = PromptBuilder::buildPanelPrompt(
        panelJson, characterRefs, sceneRefs, options);
    
    ctx.prompt = result.text;
    ctx.negativePrompt = result.negativePrompt;
    
    return true;
}

// 参考图选择 - 根据对话说话人选择
QString ImageService::selectReferenceImagePath(const GenerationContext& ctx)
{
    if (!ctx.primaryCharacterName.isEmpty() && 
        ctx.characterRefPaths.contains(ctx.primaryCharacterName)) {
        return ctx.characterRefPaths.value(ctx.primaryCharacterName);
    }
    
    if (!ctx.referenceImages.isEmpty()) {
        return ctx.referenceImages.first();
    }
    
    return QString();
}
```

### 4.5 技术要点

- **Seed 复用机制**：同一章节使用相同种子，提高一致性
- **参考图策略**：根据对话说话人选择参考图
- **指数退避重试**：API 调用失败时自动重试
- **多引擎支持**：支持 Qwen 和 VolcEngine 切换

### 5.3 实现流程

```
任务入队
     ↓
TaskQueue::enqueue()
     ├── 生成任务 ID
     ├── 设置状态为 Pending
     ├── 保存到数据库
     └── 唤醒工作线程
     ↓
TaskWorker::run() [工作线程]
     ├── 从队列获取任务

     ├── 更新状态为 Completed/Failed

```

### 5.4 关键代码

```cpp
// TaskQueue.cpp - 任务入队
QString TaskQueue::enqueue(const TaskData& task)
{
    QMutexLocker locker(&m_mutex);
    
    TaskData newTask = task;
    newTask.id = QUuid::createUuid().toString(QUuid::WithoutBraces);
    newTask.status = TaskStatus::Pending;
    newTask.createdAt = QDateTime::currentDateTime();
    
    m_tasks[newTask.id] = newTask;
    m_queue.enqueue(newTask.id);
    
    saveTaskToDatabase(newTask);
    m_condition.wakeOne();  // 唤醒工作线程
    
    emit taskEnqueued(newTask.id);
    return newTask.id;
}

// 任务执行
void TaskWorker::run()
{
    while (m_running) {
        TaskData task = dequeueTask();
        if (task.id.isEmpty()) continue;
        
        // 执行任务
        TaskResult result = executeTask(task);
        
        // 更新状态
        updateTaskStatus(task.id, result.status, result.message);
    }
}

// 失败重试
void TaskQueue::retry(const QString& taskId)
{
    QMutexLocker locker(&m_mutex);
    
    TaskData* task = getTaskRef(taskId);
    if (!task || !task->canRetry()) return;
    
    task->resetForRetry();
    m_queue.enqueue(taskId);
    saveTaskToDatabase(*task);
    
    m_condition.wakeOne();
}
```

### 5.5 技术要点

- **线程安全**：使用 `QMutex` 保护共享资源
- **持久化**：任务状态保存到数据库，重启后恢复
- **信号驱动**：使用 Qt 信号槽通知状态变化
- **并发控制**：可配置最大并发数

```cpp
// DialogueBubbleRenderer.cpp - 渲染入口
QImage DialogueBubbleRenderer::renderForPanel(const Panel& panel, const QImage& baseImage)
{
    QList<DialogueLine> dialogues = panel.dialogue();
    if (dialogues.isEmpty()) return baseImage;
    
    // 构建布局上下文
    LayoutContext context;
    context.dialogues = &dialogues;
    context.characters = &panel.characters();
    context.imageWidth = baseImage.width();
    context.imageHeight = baseImage.height();
    context.usePanelPositioning = true;
    
    // 计算气泡布局
    QList<BubbleLayout> layouts = layoutBubblesInternal(context);
    
    // 渲染到图片
    return renderLayoutsToImage(baseImage, layouts);
}

// 气泡布局计算
QList<BubbleLayout> layoutBubblesInternal(const LayoutContext& context)
{
    QList<BubbleLayout> layouts;
    QList<QRectF> occupiedRects;  // 已占用的区域
    
    for (int i = 0; i < context.dialogues->size(); ++i) {
        BubbleLayout layout;
        
        // 1. 推断角色位置
        CharacterPosition charPos = inferCharacterPosition(
            dialogue.speaker, i, *context.characters, shotType, cameraAngle);
        
        // 2. 计算气泡大小
        QRectF textRect = calculateBubbleRect(dialogue.text, maxBubbleWidth);
        
        // 3. 尝试放置（避免重叠）
        auto tryPlace = [&](qreal x, qreal y, CharacterPosition anchor) {
            QRectF rect(x, y, textRect.width(), textRect.height());
            if (intersectsAny(rect, occupiedRects)) return false;
            // ...
            occupiedRects.append(rect);
            return true;
        };
        
        // 4. 绘制气泡
        layouts.append(layout);
    }
    
    return layouts;
}

// 中文换行优化
QStringList wrapBySegments(const QString& text, const QFontMetrics& fm, int maxWidth)
{
    // 1. 按语义段分割（识别标点边界）
    QStringList segments = splitTextSegments(text);
    
    // 2. 按段填充行
    QStringList lines;
    QString currentLine;
    
    for (const QString& segment : segments) {
        if (canAppendSegment(currentLine, segment, fm, maxWidth)) {
            currentLine += segment;
        } else {
            lines.append(currentLine);
            currentLine = segment;
        }
    }
    
    return lines;
}
```

- **语义段分割**：识别标点边界，避免在词语中间断开
- **碰撞检测**：避免气泡重叠
- **角色位置推断**：根据说话人确定气泡位置
- **多气泡类型**：支持对话、心理、旁白、喊叫

```cpp
void ExportService::exportStoryboard(const QString& storyboardId, ExportFormat format)
{
    // 1. 获取所有面板

    // 2. 根据格式选择渲染器

            break;
        case ExportFormat::Webtoon:

            break;

// ExportRenderer.cpp - PDF 导出
void ExportRenderer::exportToPdf(const QList<Panel>& panels, const QString& outputPath)
{
    QPdfWriter writer(outputPath);
    writer.setPageSize(QPagedPaintDevice::A4);
    
    QPainter painter(&writer);
    
    for (int i = 0; i < panels.size(); ++i) {
        if (i > 0) writer.newPage();
        
        Panel panel = panels[i];
        QImage image = loadImage(panel.hdS3Key());
        
        // 渲染面板图片
        QRect pageRect = writer.pageLayout().paintRectPixels(writer.resolution());
        painter.drawImage(pageRect, image);
        
        // 渲染气泡
        QImage withBubbles = DialogueBubbleRenderer::renderForPanel(panel, image);
        painter.drawImage(pageRect, withBubbles);
    }
    
    painter.end();
}

// Webtoon 导出（长条图）
void ExportRenderer::exportToWebtoon(const QList<Panel>& panels, const QString& outputPath)
{
    // 计算总高度
    int totalHeight = 0;
    int width = 800;
    int gap = 20;
    
    QList<QImage> images;
    for (const Panel& panel : panels) {
        QImage image = loadImage(panel.hdS3Key());
        QImage scaled = image.scaledToWidth(width, Qt::SmoothTransformation);
        images.append(scaled);
        totalHeight += scaled.height() + gap;
    }
    
    // 合成长条图
    QImage webtoon(width, totalHeight, QImage::Format_ARGB32);
    webtoon.fill(Qt::white);
    
    QPainter painter(&webtoon);
    int y = 0;
    for (const QImage& image : images) {
        painter.drawImage(0, y, image);
        y += image.height() + gap;
    }
    
    webtoon.save(outputPath);
}
```

- **图片压缩**：ZIP 格式打包原始图片
